# 2. Market Research

**Objective**: Gather context to make the site authoritative and "Librarian-Grade".

## Steps

1.  **Competitor Audit**:
    -   Search for top 3 competitors in the Service Area.
    -   Identify their weaknesses (e.g., slow site, generic stock photos).

2.  **Entity Extraction**:
    -   Identify local landmarks, council regulations, and specific conditions (e.g., "Salt Spray", "Clay Soil", "Bushfire Zone").
    -   These entities *must* be included in the content later.

3.  **Keyword Selection**:
    -   Select 3 Primary Keywords (e.g., "Plumber Cowes").
    -   Select 3 Long-tail Questions (e.g., "How to fix blocked drain on holiday?").

4.  **Checkpoint**:
    -   [ ] `directives/master_design_prompt.md` updated with "Research Findings".
    -   [ ] User approves the Keyword Strategy.
