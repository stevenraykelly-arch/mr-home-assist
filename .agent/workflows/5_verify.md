# 5. Verify & Launch

**Objective**: Confirm the site is live, indexed, and performing.

## Steps

1.  **Push to Deploy**:
    -   `git push origin [branch_name]` (or main).

2.  **Monitor Build**:
    -   Check Coolify logs.
    -   Wait for "Deployment Successful".

3.  **SEO Audit**:
    -   Check `robots.txt` and `sitemap.xml` are accessible.
    -   Run the Schema Validation workflow: `/index_site`.

4.  **Final Polish**:
    -   Check "Contact Form" works (or placeholder is correct).
    -   Check "Explainer Video" placeholder is correct.
    -   Mobile Responsiveness verify.

5.  **Handover**:
    -   Provide User with the **Live URL**.
    -   Provide User with the **Admin/CMS** login (if applicable).
